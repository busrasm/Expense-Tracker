### ***Mini Expense Tracker***

***Açıklama:***
*Python ile geliştirilmiş, JSON tabanlı harçlık takip uygulaması.*

*Proje, sürdürülebilir bir kod yapısı için şu dosyalardan oluşur:*

*- main.py: Kullanıcı etkileşimini ve uygulama akışını yöneten ana kontrol merkezi.*

*- logic.py: Hesaplama, filtreleme ve veri işleme algoritmalarının bulunduğu iş mantığı katmanı.*

*- storage.py: JSON dosya okuma/yazma işlemlerini yürüten veri erişim katmanı.*

*- expenses.json: Uygulamanın veritabanı görevini gören veri dosyası.*
